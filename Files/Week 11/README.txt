Description
------------------------------------------------------
This project demonstrates basic Natural Language Processing (NLP)
techniques using Python's NLTK library.

Included Files:
1. week11_nltk_basic.py   - Simple example using named entity recognition.
2. week11_nltk_advanced.py - Extended analysis using the Gutenberg 'Emma' corpus.

Topics Covered:
- Tokenization
- Stopword Removal
- Frequency Distribution
- Part-of-Speech Tagging
- Lemmatization
- Named Entity Recognition (NER)
- Tree Parsing Visualization

------------------------------------------------------
Execution Instructions
------------------------------------------------------
Run each script in VS Code terminal:

> pip install nltk
> python week11_nltk_basic.py
> python week11_nltk_advanced.py

Note: On first run, NLTK will prompt you to download datasets.
Accept all downloads when asked.
